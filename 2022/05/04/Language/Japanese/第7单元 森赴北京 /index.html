第25課　これは明日会議で使う資料です

语法说明

名词前加定语

用法：

  1. [动简体]+[名]：[动简体]表习惯、将来用动词基本形；表过去用た形。
  2. [小句动简体]+[名]：[小句动简体]表习惯、将来用动词基本形；表过去用た形。
  3. [小句动简体]+[名]は + [形]/[名]です。
  4. [小句动简体]+[名]を/に/から + [动]。
  5. [小句形1/形2/名]+[名]が + [形1基本形] / [形2]+な / [名]+の 。
    注意：修饰名词的动词小句中的主语，用“が”而不是“は”。

例句：

  1. あれは 倒産した 会社です。倒産します（とうさんします）、会社（かいしゃ）那是一家倒闭的公司。
  2. これは 明日会議で使う 資料です。会議（かいぎ）、使う（つかう）、資料（しりょう）这是明天会议要用的资料。
  3. 私が明日乗る飛行機は　中国航空です。乗ります（のります）、飛行機（ひこうき）、中国航空（ちゅうごくこうくう）我明天乘坐的飞机是中国航空。
  4. 中国で買ってシーディーを　友達に貸しました。買います（かいます）、シーディー（CD）、友達（ともだち）、貸します（かします）我把在中国买的CD借给朋友了。
    中国へ転勤した友達に　手紙を書きます。転勤します（てんきんします）我给调到中国的朋友写信。
    私は知らない人から　手紙が来ました。一个不认识的人给我来了信。
  5. 入口が広いあの建物が　私の会社です。　入口（いりぐち）、広い（ひろい）、建物（たてもの）入口很大的那栋楼是我的公司。
    操作が簡単なパソコンが　欲しいです。操作（そうさ）、簡単（かんたん）我想要操作简单的电脑。
    数学が専門の先生が　休みました。数学（すうがく）、専門（せんもん）教数学的老师休息了。

[名] + のところ

用法：把不表场所的名词 -> 表场所的词。
例句：あの窓のところに　いる人は誰ですか。窓（まど）

第26課　自転車に二人で乗るのは危ないです

语法说明

动词名词化做主语

用法：

  1. [小句动简] + のは　+ [形]です：动词小句加“の” -> 使动词名词化，表示做某动作。
  2. [小句动简] + のが　+ [表示好恶、擅长与不擅长的形容词]です。

例句：

  1. 自転車に二人で乗るのは　危ないです。
  2. 李さんは絵を描くのが　好きですね。

动词名词化做宾语

用法：

  1. [小句动简]+の（＝こと）を　+ [动]。通常情况下，“の”与“こと”可以通用。
  2. 只能用“の”：句尾动词是“見ます”、“聞きます”等表示感知的动词，或“手伝います（てつだいます）帮忙”、“止めます（やめます）停止”等表示直接作用于动作对象的动词时。
  3. 只能用“こと”：句尾动词是“知らせます（しらせます）告诉”、“伝えます（つたえます）转告”、“話します”等表示语言行为的动词时。[20课 表能力、动作]

例句：

  1. 手紙を出すの（＝こと）を　忘れました。出します（だします）寄（信）忘记了寄信。
  2. 李さんは森さんが荷物を運ぶのを　手伝いました。荷物（にもつ）、運ぶます（はこぶます）小李帮森先生运行李。
  3. 李さんは森さんが休むことを　伝えました。小李转告了森先生请假的事。

　〜でしょう

用法：对自己和听话人都不能确定的事情进行推测，通常与“多分（たぶん）”一起使用。“でしょう”的简体形是“〜だろう”。
其中，用于确认对方比自己更熟悉某事，用升调；用于对方与自己意见不合，或叮问地方，用降调。

  1. [小句简体] + でしょう。
  2. [动/形1简体] + でしょう。
  3. [形2/名简体]去“だ” + でしょう。

例句：

  1. 明日の朝は大雨になる　でしょう。大雨（おおあめ）明早会下大雨吧。
  2. この本の値段は分かるませんが、多分高い　でしょう。値段（ねだん）、高い（たかい）我不知道这本书的价格，大概很贵吧。
  3. 桜の季節ですから、京都は多分にぎやか　でしょう。ぎやか　热闹。现在正式樱花观赏的季节，京都大概很热闹吧。
    1. これは多分誰かの忘れ物　だろう。这大概是谁忘记的东西吧。

[小句简体] + でしたね。

用法：表确认。
例句：日本と中国の時差は1時間でしたね。時差（じさ）日本和总过的时差是1小时来着，对吧？

　〜かもしれません

用法：表可能发生某事，比“でしょう”概率更低。 口语形式“もしかしたら、〜”，用在句首。

  1. [小句简体] + かもしれません。
  2. [动/形1简体] + かもしれません。
  3. [形2/名简体]去“だ” + かもしれません。

例句：

  1. 森さんは今日会社を休む　かもしれません。森先生今天也许不会来公司上班了。
  2. クレジットカードを落とした　かもしれません。クレジットカード credit card、落します（おとします）也许我把信用卡弄丢了。
  3. 明日は雪　かもしれません。明天也许会下雪。

　表原因

用法：

  1. それで：表示前面的句子是后面句子事态的理由；以对方叙述的事情为由，说话人用“だから”导致结果。
  2. だから：比“それで”语气强。表示前面的句子是后面句子事态的理由；也可表祈使，推测。

　つい　

用法：对意想不到的结果感到后悔、不快，多指不好的结果。
例句：バーゲン会場では、つい何でも買いたくなります。バーゲン　打折。在打折品卖场上，不知不觉什么都想买。

第27課　子供の時、大きな地震があります

语法说明

[小句简体] + 時

用法：表时间的名词短语。

  1. [动基本]（〜する） + 時：表现在将来；习惯性动作。
  2. [动た]（〜した） + 時：过去式。
  3. [动ている]（〜ている） + 時：现在进行式。[第15、16课 “动ている”表示动作正在进行，动作的结果、状态]
  4. [形1]+ 時。
  5. [形2] + な + 時, [名] + の + 時。

例句：

  1. 日本に行く時、たくさんお土産を買いました。
  2. 日本に行った時、たくさんお土産を買いました。
  3. 忙しい時、家族みんなで仕事をします。
  4. 私は毎日散歩しています。我每天散步。
  5. a. 暇な時、私は町で買い物をしました。町（まち）街道
    b. 供の時、大きな地震がありました。

[动词词干] + ながら + [动]

用法：表同一主体同时进行两个动作，其中，后面动作是主要动作。
例句：李さんはテレビを見ながら食事をしています。食事（しょくじ）吃饭

[名] + で

用法：表原因、理由。
例句：仕事で、楊さんと会っていたんですよ。仕事（しごと）因工作我与杨先生见面来着。

话说...

用法：

  1. そう言えば（そういえば）：将要说的话与之前的内容有点联系。
  2. ところで：完全开始不一样的话题。

例句：

  1. そう言えば、小さい時、よく祖母と一緒に公園へ行きました。祖母（そぼ）

第28課　馬さんは私に地図をくれました

语法说明

授受关系

用法：授受关系均包含感恩的情感。

  1. 名词的授受关系：第8课“〜を あげます/もらいます”。
    a. [名，人A]は　[名，人B]に　[名，物品]を　あげます。A给B某物，“あげます”相当于“给”。
    b. [名，人A]は　[名，人B]に　[名，物品]を　もらいます。A接受B的某物，“もらいます”与“あげます”相反。
    c. [名，人B]に　[名，物品]を　くれる。别人给自己。
  2. 动词的授受关系：[动て]　あげます/もらいます。
    a. [动て] あげます：我帮别人，用于平辈、关系好的人 -> 礼貌：[动词词干] + ましょうか。
    b. [动て] もらいます：我被别人帮，让别人做。
    c. [动て] くれる：别人帮我。委托别人帮自己做某事，“〜くれませんか”比“〜くれますか”更礼貌。

例句：

  1. 名词
    a. 私は　小野さんに　お土産を　あげます。我给小野礼物。
    b. 私は　小野さんに　辞書を　もらいます。我从小野那儿得到一本词典/小野给了我一本词典。（じしょ）
  2. 动词
    a. 森さんはお年寄りの荷物を持って　あげました。お年寄り（おとしより）森先生帮老人拿了行李。
    b. 林さん、もう少し待って　もらいたいのですが。林先生，我想请你再等等。
    c. 友達が面白い本を教えて　くれました。朋友给我介绍了一本有趣的书。
