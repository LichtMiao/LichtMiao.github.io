第29課　電気を消せ

语法说明

[动命令]

用法：说话人对听话人下命令。命令形多用于紧急情况；观看体育比赛；上司对下属；关系好的朋友之间；直接引用或解释词语。因感情太过强烈，平常一般不使用。

  1. 动词基本形“う”段 -> “え”段。
  2. 动词基本形“る” -> “ろ”。
  3. 来る（くる）-> 来い（こい）,する -> しろ，〜する　-> 〜しろ。

例句：
電気を消せ。電気（でんき）关灯！

比[动命令]稍客气的说法

用法：

  1. [动词词干] + なさい：表肯定，用于老师对学生，或父母对孩子提出要求。
  2. [动基本] + な：表否定，禁止某人做某事。

例句：

  1. 質問に答え なさい。質問（しつもん）、答う（こたう）回答问题。
  2. ここに車を止める な。

请求某人做某事

用法：祈使句，一般用于亲密的人之间。

  1. [动て]：表肯定。
  2. [动ない] + で：表否定。

例句：

  1. もうちょっと　急いで。再稍微快一点。
  2. 遠慮しない で。遠慮（えんりょ）别客气。

[个体] + という + [群体]

用法：“叫做…的…”。
例句：

  1. それはフジ という 花です。フジ　紫藤，那是一种叫紫藤的花。
  2. 田中　という　名前は日本人の名前です。田中这个姓是日本人的姓。

Aは + [名]/[小句] + という + Bです

用法：对A下定义，B多为“意味”等词语。
例句：
このマークはタバコを吸うな　という　意味です。这个符号是禁止吸烟的意思。

第30課　もう11時だから寝よう

语法说明

[动意志]

用法：表意志，决心；提议。意志形是17课“[动词词干] + ませんか/ましょう”的简体形式。

  1. 动词基本形“う”段 -> “お”段+“う”。
  2. 动词基本形“る” -> “よろ”。
  3. 来る（くる）-> 来よう（こよう）,する -> しよう，〜する　-> 〜しよう。

例句：
もう11時だから、寝よう。[动2意志]

[动意志]的使用

用法：

  1. [动意志形] + と思います：比单纯用意志形更加礼貌。当下的意志。
  2. [动意志形] + と思っています：自己的某种意志持续了一段时间。较犹豫；转述他人。

例句：

  1. 今日、会社を休もう　と思います。我今天不想上班。
  2. 明日病院へ行こう　と思っています。明天我想去医院。

そろそろ

用法：

  1. そろそろ：表示做某事或某事态发生的时间迫近。
  2. そろそろ + ましょうか：提醒或催促对方和自己一起行动。
  3. そろそろ + [动意志] + か：提醒或催促对方和自己一起行动。

例句：

  1. そろそろ食事の時間ですよ。食事（しょくじ）
  2. そろそろ失礼します。去别人加做客，发觉该回家了。
  3. そろそろ出発しようか。[动3意志] 准备出发吧。

楽しみです

用法：

  1. 楽しみです（たのしみです） = 楽しみにしています。表示盼望、期待的心情。
  2. 楽しみにしていてください。请你期待。

[场所/容器]で + いっぱい

用法：表某处/容器被充满。
例句：
この季節は、どの行楽地も人で いっぱいですよ。行楽地（こうらくち）景点

第31課　このボタンを押すと、電源が入ります

语法说明

[动词基本形/ない形]、...

用法：
“〜と”用在表示恒常性状态、真理、反复性状态、习惯等内容的复句里，表示前面句子是后面句子的条件。
例句：

  1. このボタンを押すと、電源が入ります。電源（でんげん）按下这个按钮，电源就接通了。[恒常状态]
  2. 食べ物を食べないと、人間は生きることができません。人間（にんげん）不吃食物，人类就无法活。[真理]
  3. 私は、その店へ行くと、いつもコーヒーを飲みます。我去那家店的时候总是点咖啡。[反复状态]
