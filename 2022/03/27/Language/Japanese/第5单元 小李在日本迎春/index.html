第17課　私は　新しい　洋服が　欲しいです。

语法说明

[名]が　欲しいです。

用法：表达愿望时使用，“想要~”。
例句：私は　新しい　洋服が　欲しいです。（ようふく）

[名]を/が　[动词词干]たいです。

用法：[动词词干] 是动词“ます”形去掉“ます”后留下的部分。

  1. “想做~”。
  2. “を/が”可以通用。特殊地，“欲しいです”前面只能加“が”。

例句：私は　映画を/が　見たいです。見る -> 見ます[动2]

“想要”的全面肯定/否定

用法：

  1. 全面肯定：[疑问词] + でも + 肯定句。表示任何情况下都相同。
  2. 全面否定：
    [名]を + [动] -> [疑问词] + も + [动词否定]。
    [名]に/から/と + [动] -> [疑问词] + にも/からも/とも + [动词否定]。　
    [名]へ + [动] -> [疑问词] + も/へも + [动词否定]。

例句：
a. 誰でも　いいです。谁都可以。
b. 何も　飲みたくないです。什么也不想喝。
c. 誰にも　会いたくないです。谁也不想见。
d. どこへも　行きたくないです。哪里也不想去。

表示提议、邀请

用法：

  1. [动词词干] + ませんか。动词的否定形式“〜ません”加表示疑问的“か”，更加礼貌。
  2. [动词词干] + ましょう。[动词意志形，第30课]

例句：
a.
一緒に　お茶を　飲みますんか。一起喝茶好吗？（いっしょ）/少し　休みませんか。稍微休息一下怎么样？
b. そろそろ　行きましょう。该走了。そろそろ　快要

第18課　携帯電話は　とても　小さく　なりました。

语法说明

自动词和他动词

用法：

  1. 不带宾语，主语用“が”表示。不考虑外力影响的情况下，自主地动作或发生变化。如“なる/なります”。
  2. 带宾语，宾语用“を”表示。有主语作用于宾语而使其发生变化。如“する/します”。

例句：
a. 窓を　開きました。窗户自己开了。（まど）
b. （李さんが） 窓を　開きました。小李打开了窗户。

[形]/[名]表达变化

用法：

  1. [形1]去“い” + く なります/なる。なる [动1，自动词，自己在变]，表性质或状态的变化。
  2. [形1]去“い” + く します/する。する [动3，他动词，他人使能而变]，因主语的意志性动作、作用而引起事物发生变化的场合。
  3. [形2]/[名] + に なります/なる。なる [动1，自动词，自己在变]，表性质或状态的变化。
  4. [形1]/[名] + に します/する。する [动3，他动词，他人使能而变]，因主语的意志性动作、作用而引起事物发生变化的场合。

例句：
a. だんだん　暖かく　なります。天气渐渐变暖。暖かい（あたたかい）/パソコンは　安く　なりました。电脑变便宜了。
b. ジュースを　冷たく　します。把果汁冰镇一下。冷たい（つめたい）
c. もう　元気に　なりました。已经恢复健康了。
d. 部屋を　綺麗に　してください。请把房间打扫干净。（きれい）（[动词て形] + ください，请求某人做某事。）

两个中哪个更好

用法：比较两个以上事物时，认为其中一个更好。

  1. [形1] + ほうが　いいです。
  2. [形2]＋な + ほうが　いいです。
  3. [动词た形] + ほうが　いいです。
  4. [名]の + ほうが　いいです。

例句：
a. 旅行の　荷物は　軽い　ほうが　いいです。（かるい）
b. 子供は　元気な　ほうが　いいです。
c. もっと　野菜を　食べた　ほうが　いいです。还是多吃点蔬菜好。（やさい）
d. 速いですから、飛行機の　ほうが　いいです。

与~相称

用法：

  1. [人]は [物]が　似合います。（[自动词]，にあいます）
  2. [物]は [人]に　似合います。

例句：
a. （〜が）よく　似合いますよ。/浴衣が　似合います。
b. 父に　似合いません。

一会儿，不久

用法：

  1. もうすぐ：更常用。
  2. 間もなく（まもなく）：更正式。

更加

用法：
3. もっと：较随便。
4. さらに：更正式。

さあ

用法：

  1. 用于向对方提议，或敦促对方做某事。
  2. 我不知道。

第20課　スミスさんは　ピアノを　弾く　ことが　できます。

语法说明

[人]は + [动词基本形] +　ことが　ですます。

用法：表示能力。肯定回答用“できます”，否定回答用“できません”。
例句：スミスさんは　ピアノを　弾く　ことが　できます。（ひく）

[名]は + [动词基本形] + ことです。

用法：谓语部分的内容是一种动作、行为时使用。
例句：私の　趣味は　切手を　集める　ことです。我的爱好时集邮。（しゅみ）（きって）（あつめる）

[动词基本形] + こと

用法：可以起到名词的作用。
例句：小野さんは　車を　運転する　ことが　できません。小野不会开车。（うんてん）

表示动作的前后关系

用法：

  1. [动词基本形] / [名]の 前に（まえに）：前に表示一个动作发生在另一个动作之前。
  2. [动词た形] / [名]の　後で（あとで）：表示一个动作发生在另一个动作之后。[第21课]

例句：
a. 毎日　寝る　前に　シャワーを　浴びます。（あびます）
b. 映画を　見た　後で、食事を　しました。（しょくじ）
c. 仕事の　後で、映画を　見ます。（しごと）

疑问词 + か

用法：

  1. いつ/どこ/誰 + か：表示某时/某地/某人。
  2. いつ/どこ/誰/何 + か：本身并不表示疑问的意思，也可以用于陈述句。

例句：

  1. 部屋に　誰か　いますか。房间里有人吗？
  2. 冬休み、どこかへ　行きたいです。寒假想去个地方。
