第17課　私は　新しい　洋服が　欲しいです。

语法说明

[名]が　欲しいです。

用法：表达愿望时使用，“想要~”。
例句：私は　新しい　洋服が　欲しいです。（ようふく）

[名]を/が　[动词词干]たいです。

用法：[动词词干] 是动词“ます”形去掉“ます”后留下的部分。

  1. “想做~”。
  2. “を/が”可以通用。特殊地，“欲しいです”前面只能加“が”。

例句：私は　映画を/が　見たいです。見る -> 見ます[动2]

“想要”的全面肯定/否定

用法：

  1. 全面肯定：[疑问词] + でも + 肯定句。表示任何情况下都相同。
  2. 全面否定：
    [名]を + [动] -> [疑问词] + も + [动词否定]。
    [名]に/から/と + [动] -> [疑问词] + にも/からも/とも + [动词否定]。　
    [名]へ + [动] -> [疑问词] + も/へも + [动词否定]。

例句：
a. 誰でも　いいです。谁都可以。
b. 何も　飲みたくないです。什么也不想喝。
c. 誰にも　会いたくないです。谁也不想见。
d. どこへも　行きたくないです。哪里也不想去。

表示提议、邀请

用法：

  1. [动词词干] + ませんか。动词的否定形式“〜ません”加表示疑问的“か”，更加礼貌。
  2. [动词词干] + ましょう。[动词意志形，第30课]

例句：
a.
一緒に　お茶を　飲みますんか。一起喝茶好吗？（いっしょ）/少し　休みませんか。稍微休息一下怎么样？
b. そろそろ　行きましょう。该走了。そろそろ　快要
