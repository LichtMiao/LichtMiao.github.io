动词变形汇总

Posted on 2022-03-22 | In Language , Japanese
动词

[动2]
[动3]
基本形 “う”段 いる（“一る”）
える（“二る”）		 来る
する
〜する（前面加汉字词或外来词，
也可以说是“[名]を+する”的简写）
例 書く
急ぐ（いそぐ）
飛ぶ（とぶ）、読む、死ぬ
待つ、売る（うる）、買う（かう）
話す（はなす）		 起きる、見る
食べる、寝る		 来る
する
勉強する
命令形 “う”段 -> “え”段 いる（“一る”）-> いろ
える（“二る”）-> えろ		 来る -> 来い（こい）
する -> しろ
〜する -> 〜しろ
例 書け
急げ（いそげ）
飛べ（とべ）、読め、死ね　
待て、売れ（うれ）、買え（かえ）
話せ（はなせ）		 起きろ、見ろ
食べろ、寝ろ		 来る -> 来い（こい）
する -> しろ
勉強しろ
ます形 变成“い”段+ます 去“る”+ます 来る -> 来ます
する -> します
去“する”+します
例 書きます
急ぎます
飛びます、読みます、死にます
待ちます、売ります、買います
話（はなし）ます		 起きます、見ます
食べます、寝ます		 来る -> 来ます
する -> します
勉強します
动词词干：去掉ます后留下的部分 書き
急ぎ

待ち、売り、買い
話（はなし）		 起き、見
食べ、寝		 来
し
勉強し
て形 ます形去“ます”+て，且发音发生变化：
き -> いて
ぎ -> いで
び、み、に -> んで
ち、り、い -> って（特殊：行き -> 行って）
し -> して		 ます形去“ます”+て ます形去“ます”+て
例 書いて
急いで
飛びんで、読んで、死んで
待って、売って、買って
話して		 起きて、見て
食べて、寝て		 来て

勉強して
た形 て形去“て”+た
て形去“で”+だ		 て形去“て”+た
て形去“で”+だ		 て形去“て”+た
て形去“で”+だ
例 書いた

ない形 ます形去“ます”，且最后一个音变成“あ”段+ない。特殊地，如果最后一个音是“い”段，变成“わ”+ない： ます形去“ます”+ない 来ます（きます）-> 来ない（こない）
します -> しない
〜します -> 〜しない
