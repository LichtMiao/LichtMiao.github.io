发音规则

ㅂ/ㅈ/ㄷ/ㄱ 略送气

ㅂ [b]/ㅈ [j/z]/ㄷ [d]/ㄱ [g] 放在词首略送气 = [p]/[c]/[t]/[k]

ㅂ 的音变

ㅂ + ㄴ = [mi] ㅁ

ㅎ 的音变

ㅎ 放在词中、句中可以淡化、轻读，甚至不发音。

ㅇ 的音变

ㅇ 放在左边、上边不发音；放在下面发 ng 音。

ㅂ/ㅈ/ㄷ/ㄱ 的音变

  1. ㅂ/ㅈ/ㄷ/ㄱ + ㅎ = ㅍ/ㅊ/ㅌ/ㅋ 发送气音
  2. ㅎ + ㅂ/ㅈ/ㄷ/ㄱ = ㅍ/ㅊ/ㅌ/ㅋ

ㅅ 的音变

ㅅ + ㄷ/ㅎ = ㅌ

ㅅ/ㅇ/ㅁ 的音变

  1. ㄱ + ㄴ/ㅇ = ㅇ + ㄴ/ㅇ
    각년 [강년]
  2. ㅇ/ㅁ + ㄹ = ㅇ/ㅁ + ㄴ
    양력 [양녁]

ㅣ 的音变

  1. ㅅ/ㅆ + ㅣ = [xi]
  2. ㅈ/ㅉ + ㅣ = [ji], ㅊ + ㅣ = [chi]

ㅡ 的音变

  1. ㅇ + ㅡ = [eu] 读 ㅡ 本身的音
  2. ㅂ/ㅃ/ㅍ/ㅁ + ㅡ = 브 [bu]/쁘 [bu]/프 [pu]/므 [mu]，ㅡ 读小 ㅜ 的音
  3. 其他辅音 + ㅡ = 读辅音本身的音

ㅢ 的音变

  1. ㅢ 放在词首，读本身的音 [eui]
  2. ㅢ 不放在词首，读音为 [i]
  3. ㅢ 含义为“~的”时，发音为 [e]。너의 [neo e]你的
  4. 其他辅音 + ㅢ = ㅢ 读音为 [i]

ㅒ/ㅖ 的音变

  1. ㅇ + ㅒ/ㅖ = [yae]/[ye] 读 ㅒ/ㅖ 本身的音
  2. 其他辅音 + ㅒ/ㅖ = [ae]/[e]

双收音

只有辅音可以做收音，包括单收音和双收音。
双收音只读一边的音。

  1. 右边音：ㄹㄱ/ㄹㅍ/ㄹㅁ [lg，老公]/[lp，老婆]/[lm，老妈] 读右边音 [g]/[p]/[m]
  2. 左边音：其他双收音全部读左边音。
  3. 特殊单词：ㄹㅂ [lb，老爸] 在词汇 밟다 [pab da] 中读右边音。

连化音

当收音（除了 ㅎ/ㅇ）和后面以元音为开头的音节相连时，要将收音移到后一个音节与元音相拼。

紧化音

ㅂ/ㄷ/ㄱ + 辅音ㅂ/ㅈ/ㄷ/ㄱ/ㅅ = 辅音发音为 ㅃ/ㅉ/ㄸ/ㅆ

  1. ㅇ + ㅒ/ㅖ = [yae]/[ye] 读 ㅒ/ㅖ 本身的音
  2. 其他辅音 + ㅒ/ㅖ = [ae]/[e]

腭化音

  1. ㅌ + 이 = 치 [qi]
  2. ㄷ + 히 = 치 [qi]
  3. ㄷ + 이 = 지 [ji]
