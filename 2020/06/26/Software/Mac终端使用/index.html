Mac终端使用

投稿日 2020-06-26 | In Software

Command + q: 退出

向上滑：两指向上

防止程序遮挡：四指头向上推打开“调度中心”，也可新建桌面

将文件夹拖到dock竖线右侧，实现文件的快速访问

文件保存：默认保存在“文稿”中，保存时若需要专门文件夹，需要下拉箭头，存入相应路径。“标记”功能：对文稿进行分类。

选中文件：command+a选中所有，按住command+选中文件

文件比较深，可使用分栏方式查看文件

点击空格快速预览文件（可跨越文件类型），而不用打开

双指按触控板，调出右键选项

选中文件，按回车，重命名

对文件的拷贝、粘贴、删除：

拷贝：****command+c****；拖拽时****+option****（光标变成了绿底白色加号），移动到对应文件夹****

粘贴：****command+v****

Mac****中的拖动默认是剪切；没有直接剪切的快捷键，或者：****command+c****，然后****option+command+v****

删除：****command+delete****（直接点****delete****没有效果）****

中英文切换：****control+****空格****

截图：****shift+command+4 ;****截屏：****shift+command+3****

对文字的操作与win相同

还原：command+z（基本所有操作都可还原上一步操作）

搜索功能：command+空格。双击即打开，command+双击：显示文件所在的文件夹

若启动台被移除了，也可直接搜索，拖到dock即可。

开启三指拖动：偏好设置->辅助功能->键盘和鼠标，不用点击文件，只需要三指直接对文件进行拖动。

安装程序.dmg

下载的文件压缩包一般刚在桌面，在弹出的对话框中将这个压缩包拖到Appliaction就可安装程序。

删除文件：只有从app store中下载的程序才可以点叉删除；其他文件的删除：访达->应用程序，将不需要的程序直接拖到废纸篓。桌面的安装包删除：双击点推出。

NTFS tuexar：在Mac中兼容磁盘格式用的

有些软件由于未按apple的标准来开发，打开的时候会提示文件已损坏或未知来源而打不开，系统偏好设置->安全与隐私->在通用中解锁任何来源的选项 在终端中输入sudo spctl —master-disable，然后输入密码解锁隐藏的任何来源选项。

智能缩放：双指轻拍两下

旋转：双指旋转

双指从右边进入，显示通知中心

浏览网页的时候，双指左右扫，进行网页返回与前进

在全屏应用程序和普通桌面之间进行切换：四指左右扫

调度中心平铺所有打开的程序和桌面：四指向上扫

对当前使用的一个程序的几个页面进行平铺：四指向下扫

显示启动台：拇指和三个手指合拢

显示桌面：拇指和三个手指分开

关闭当前程序的所有页面：option+关闭

查看每个文件的详细信息：option+command+i

修改文件默认打开方式：双击下拉菜单中可以修改一次的打开方式；option+修改打开方式，可以永久修改默认打开返回

长文件的定位：option+箭头点击，就可以定位到任何想到达的地方

展开文件夹树：option+倒三角箭头

control+command+空格：调出emoji

播放器：iina，它还可通过粘贴网址来播放网页上的视频

压缩软件：keka

欧路词典

Cheatsheet：长按command不松，显示当前应用的所有快捷键

日历：itsycal mowglii.com/itsycal/

amphetamine：阻止进入休眠

如果你想将当前 command line 会话切换到其他目录，需要用到三个命令：pwd，ls和cd。

pwd：print working directory，它会显示当前目录的绝对路径。

ls：list directory contents，它会列出当前目录的内容。这个命令还有其他参数可选。
cd：change directory，它会改变当前目录到你指定的目录。如果你不指定，则会返回你的 home folder。

pwd　　　　　　当前工作目录

ls 列出当前目录下所有文件

cd（不加参数）　　进root

cd ~ 返回root

cd（folder）　　进入文件夹 //cd MyBlog/source/_posts

cd ..　　　　　　上级目录

cd -　　　　　　返回上一次访问的目录

rm 文件名 　　　　删除

cat 文件名(|less)　　在终端下查看文件

cp 文件名 目标目录　　将文件拷贝到目标目录下

mkdiv　　　　　　新建文件夹

g++ 源文件名　　　　编译源文件，产生a.out

./文件名　　　　　　运行 例如：./a.out < 输入文件名 > 输出文件名

control+d　　　　　中断a.out运行

nano 　　　　　　编写脚本语言　　ctrl+o存储

nano ….sh　　　　打开

bash ….sh　　　　运行脚本

echo “…$i…”　　　输出语句

tar -zxf abc.tar.gz tar文件解压

ssh root@192.168.1.222 以root账号远程连接222服务器

unrar x abc.rar rar文件解压，需要安装rar工具

open /Users/sunww 使用 open 命令，如打开/Users/sunww 文件夹
